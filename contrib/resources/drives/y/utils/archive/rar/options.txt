   Command line options


  Usage:     RAR <command> -<switches> <archive>
                 <@listfiles...> <files...>

   Commands:

  a             Add files to an archive
  c             add a Comment to an archive
  cf            add Comments to Files
  cw            Write a Comment from an archive to a file
  d             Delete files from an archive
  e             Extract files to the current directory
  en            ENter to an archive
  f             Freshen files within an archive
  k             locK an archive
  l             List the contents of an archive
  m[f]          Move to an archive [Files only]
  p             Print file to stdout
  r             Repair an archive
  rr[N]         Add a data recovery record
  s[file.sfx]   Convert an archive to SFX [alternate SFX module may be used]
  t             Test archive files
  u             Update files within an archive
  v             Verbosely list the contents of an archive
  x             eXtract files with full path

   Switches:

  -$[drive]     provides volume label support
  -ac           Clear Archive attribute after compression
  -ao           add files with Archive attribute set
  -ap           set path inside archive
  -av           add Authenticity Verification (registered versions only)
  -av-          disable checking or adding authenticity information
  -b            enable Backup archives
  -bw           Black & White (monochrome) mode
  -c-           disable the display of Comments
  -cfg-         disable read ConFiGuration
  -col          COLor mode
  -dh           disable sharing files
  -ds           do not sort files while adding to a solid archive
  -e<s,h,a,r>   set file Exclude attributes
  -ed           do not add Empty Directories
  -ee           disable the saving of extended attributes (OS/2 version only)
  -ems-         disable EMS usage (DOS version only)
  -ep           Exclude Paths from names
  -ep1          Exclude base dir from names
  -f            Freshen files
  -ialt         only right <Alt> key changes help info in bottom line
  -m<0..5>      set compression level (0-store...3-default...5-maximal)
  -md<size>     select dictionary size in Kb (non-DOS versions only)
  -mm[f]        multimedia compression [force]
  -o-           do not Overwrite existing files
  -o+           Overwrite existing files
  -p<password>  set Password
  -r            Recurse subdirectories
  -r0           Recurse subdirectories (wildcards only)
  -ri<P>[:<S>]  set priority and sleep time (OS/2 version only)
  -rr[N]        add data recovery record (N - number of sectors, up to 8)
  -s[e,N,v]     create a Solid archive
  -sfx[fil.sfx] create SFX archives (alternate SFX module may be used)
  -std          STDout mode
  -tk           do not modify the archive date
  -tl           set the archive date as the date of the newest file
  -u            Update files
  -v            create Volumes with size autodetection
  -v<size>      create Volumes:  Size=<size>*1000 bytes
  -v<size>b     create Volumes:  Size=<size> bytes
  -v<size>k     create Volumes:  Size=<size> kilobytes
  -v<size>f     create Volumes:  <size> can be 360, 720, 1200, 1440, 2880
  -vd           erase disk contents before creating volume
  -w<path>      assign a Work directory
  -x<file>      eXclude the specified File
  -x@<list>     eXclude files in the specified List file
  -y            assume Yes on all queries
  -z<file>      add a comment to an archive from the file


    The SFX archive module recognizes the following switches:

  -e            Extract from the SFX archive (default)
  -t            Test the SFX archive
  -v            View archive contents


   Interactive mode


  Usage:

         RAR

  Options:

   Alt-A  Select dictionary size (OS/2 version only)
   Alt-C  Toggle between Color and monochrome mode
   Alt-D  Select Drive
   Alt-J  DOS shell
   Alt-M  Select compression Method
   Alt-P  Set Password
   Alt-R  Set priority and sleep time (OS/2 version only)
   Alt-S  Save Setup
   Alt-W  Set Work directory for temporary files


  Directory functions:


   F1       Help information

   F2       Add file(s) to an archive, if the archive does not exist
            it will be created

   F3       View a file (if external viewer is not set, the
            internal viewer will be used)

   F4       Freshen files within an archive - only recently changed files
            will be added and will replace older files existing in
            the archive

   F5       Create archive volumes

   F6       Add files not already in the archive and update those
            files changed since they were packed to the archive

   F7       Move file(s) to an archive

   F8       Delete file or directory

   F9       Set up / Save options

   F10      Quit RAR. ESC can also be used to perform the same
            function.

   Alt-F2   Create a solid archive.

   Alt-F3   View file (with internal viewer).

   Alt-F4   Create a SFX archive.

   Alt-F5   Create a SFX-volume.

   Alt-F6   Create a solid archive volumes.

   Alt-F7   Create a solid SFX-volume.

   Alt-F8   Repair a broken archive.

   Alt-F9   Create a solid SFX archive.


  Archive functions:


   F1       Help information

   F2       Test an archive

   F3       View a file (if an external viewer is not set,
            the internal viewer will be used)

   F4       Extract files from an archive with directory
            structure

   F5       Add comment to an archive

   F6       Extract file(s) to the current directory

   F7       Make an SFX-archive

   F8       Delete file(s) from an archive

   F9       Set up / Save options

   F10      Return from an archive. ESC can also be used to
            perform the same function.

   Alt-F3   View a file (with internal viewer)

   Alt-F4   Extract file(s) to alternate path

   Alt-F5   Add file comment(s)

   Alt-F6   Add archive recovery record

   Alt-F7   Lock archive to prevent modifications
