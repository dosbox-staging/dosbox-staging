# Glad 2

These OpenGL loaders have been generated by the online version of Glad 2, the
multi-Language Vulkan/GL/GLES/EGL/GLX/WGL loader-generator.

Check out the permalink to the online generator in `include/glad/gl.h` if you
want to re-generate the same loaders.
